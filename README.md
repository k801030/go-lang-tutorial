# go-lang-tutorial


## Some Commands

### How to run

```
make 
```

### How to test

```
make test
```

### How to call APIs

```
curl -s -o /dev/null -w %{http_code} "http://localhost/health-check"
```

```
curl -s "http://localhost:/user/vison"
```
